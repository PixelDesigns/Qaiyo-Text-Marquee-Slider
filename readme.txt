Testreszabható, animált marquee slider sorokkal, ikonokkal, elválasztókkal és halványuló szélekkel.

== Description ==

A Qaiyo Text Marquee Slider egy könnyen használható WordPress plugin, amellyel gyönyörű, végtelen gördülő szövegsorokat hozhatsz létre — hasonlóan a modern SaaS weboldalak ticker sávjaihoz.

**Főbb funkciók:**

* 1, 2 vagy 3 sor beállítása
* Soronként egyedi irány (balra vagy jobbra)
* Egér ráhúzásra megáll az adott sor, a többi megy tovább
* Tetszőleges számú szövegelem soronként
* Egyedi szövegszín, betűtípus és linkelés elemenként
* Ikon kép szöveg előtt és/vagy után (PNG, JPG, WebP, SVG)
* Egyedi távolság szövegelemek között
* Soronkénti elválasztó kép a szövegelemek között
* Halványuló szélek (fade effekt)
* Egyedi sorok közötti távolság
* Kategorizálás és lomtár támogatás
* Többnyelvű: angol, német, francia, spanyol, magyar
* Kompatibilis: PolyLang, TranslatePress, WPML

== Installation ==

1. Töltsd fel a `qaiyo-text-marquee-slider` mappát a `/wp-content/plugins/` könyvtárba.
2. Aktiváld a plugint a WordPress admin felületén.
3. Menj a „Marquee Slider" menüpontra és hozz létre egy új slidert.
4. Illeszd be a shortcode-ot bármelyik oldalra: `[qaiyo_text_marquee id="123"]`

== Frequently Asked Questions ==

= Hogyan illesztem be a slidert? =
Használd a `[qaiyo_text_marquee id="123"]` shortcode-ot, ahol a 123 a slider ID-je.

= Hány sort állíthatok be? =
Maximum 3 sort, mindegyik egyedi iránnyal és sebességgel.

= Milyen képformátumokat támogat? =
PNG, JPG, WebP és SVG.
